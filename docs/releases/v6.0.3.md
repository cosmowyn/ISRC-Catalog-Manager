# ISRC Catalog Manager 6.0.3

[![Release](https://img.shields.io/github/v/tag/cosmowyn/ISRC-Catalog-Manager?label=release)](https://github.com/cosmowyn/ISRC-Catalog-Manager/releases/tag/v6.0.3)

Version: 6.0.3
Date: 2026-06-03
Type of update: Patch

## Fixes
- Fixed a profile-switch crash path after the encrypted database password prompt.
- Profile teardown no longer refreshes visible workspace docks while profile-scoped services have already been cleared.
- Profile-switch activation now catches and reports finalization callback failures instead of letting Qt callback exceptions escape.
- Added regression coverage for switching to an encrypted profile after entering its password.

## Update Notes
- This release supersedes v6.0.2 for automatic updates.
- Existing encrypted profiles remain valid. Reopen or switch to them with the password chosen during profile creation.
- If the password prompt is cancelled during profile switching, the current profile remains open.

## Validation Focus
- Reproduced the encrypted profile-switch path through the app-shell profile selector.
- Added an app-shell regression for unlocking and switching to an encrypted profile through the password prompt.
- Re-ran focused profile/session/security tests, UI Help QA, the `ui-app-workflows` grouped shard, compile, Ruff, Black, and mypy.
